◆ dirt_field_info_alter()

dirt_field_info_alter ( &  $info)

Implements hook_field_info_alter().

Adds privacy options to field properties:

  • designates some fields as admin-only (for edit and/or view operations),
  • designates some fields as private, meaning visible to admins and survey author but no one else (including other participants).

Note that admin-only fields are necessarily private (even if not explicitly set as private in field settings).

Parameters
[in,out]$infoArray of information on field types exposed by hook_field_info() implementations.
