dirt_get_field_group_basenames (   $modes = array('form', 'default'))

Returns an associative array of field group basenames => info including bundle name and mode(s). Bundle name "ALL_SURVEYS" indicates that this group should be attached to all survey content types.

Parameters
[in]$modesOptional View modes (defaults to form and default).

◆ dirt_get_field_group_info()

dirt_get_field_group_info ( )

Returns array of field groups where each array item is an array with keys:

'basename' : group basename (without mode) 'bundle' : bundle group is attached to 'mode' : mode (form or node view),

using definitions from dirt_get_field_group_basenames().

This function effectively enumerates all field groups installed in this module.

Return values
Arrayof arrays with keys 'basename', 'bundle', and 'mode'.
