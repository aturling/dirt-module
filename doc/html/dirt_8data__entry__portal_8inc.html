◆ dirt_get_all_possible_survey_dates()

dirt_get_all_possible_survey_dates (   $uid)

Get oldest survey in system from $user and generate array of every date in DIRT short survey date format (month+year) from collection start date to current date.

If no start date manually set in user profile, assume date of oldest collection start date.

If no surveys in system, return single entry of current month+year.

Parameters
[in]$uidUser ID (uid).
Return values
Arrayof survey dates in DIRT short survey date format.

◆ dirt_get_scheduled_surveys()

dirt_get_scheduled_surveys (   $uid)

Return array of associative arrays of every scheduled survey between collection start date and present date.

Return array has form: array( array( 'type' => Survey type machine name string, 'date1' => Oldest possible date of survey, 'date2' => Most recent possible date of survey, ), ) where 'date1' and 'date2' specify the date range for the scheduled survey (e.g., 12 month span for once a year surveys, 1 month span for monthly surveys, etc.)

Parameters
[in]$uidUser ID (uid).
Return values
Arrayof associative arrays of scheduled survey info .
